Опции панели спавна сущностей
Default спавнит малые сущности (например, кружки) без выравнивания по сетке, при этом выравнивая блочные сущности (например, стены) по сетке.
PlaceFree спавнит все сущности без выравнивания.
PlaceNearby ограничивает радиус спавна примерно 2 плитками.
SnapgridCenter выравнивает сущность по центру плитки.
SnapgridBorder выравнивает сущность по границе плитки.

Опции выравнивания по сетке
AlignSimilar привязывает сущность к похожим сущностям. В настоящее время сломано.
AlignTileAny выравнивает сущность по сетке.
AlignTileEmpty целевая плитка должна быть пустой.
AlignTileNonDense в целевой плитке не должно быть коллайдеров.
AlignTileDense коллайдеры должны быть в целевой плитке.
AlignWall привязывается к вертикальным полуплиткам.
AlignWallProper выравнивает сущность по центру краев плитки.
